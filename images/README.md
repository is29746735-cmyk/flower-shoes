# 꽃신 사진 폴더

이 폴더에는 홈페이지에서 사용할 사진 파일을 둡니다.

## 갤러리 사진 (현재 data.js 가 가리키는 파일)

| 파일 이름 | 내용 |
|---|---|
| `interior-01.webp` | 카페 입구 진열대 — 수공예 꽃신과 라탄 가방 |
| `interior-02.webp` | 라탄 핸드백과 전통 꽃신 가까이 |
| `interior-03.webp` | 통창 좌석과 정원 풍경 |
| `interior-04.webp` | 격자 우드 천장 메인 홀 |
| `interior-05.webp` | 한옥 분위기 룸 |

## 기타

| 파일 이름 | 내용 |
|---|---|
| `logo.png` | 꽃신 로고 (시작 화면 좌상단 박스에 자동 표시) |

## WebP 로 변환하는 방법

### 방법 ① — 같은 폴더의 `convert.py` 사용 (Python 필요)
1. 원본 사진(JPG / PNG / HEIC 등)을 이 `images/` 폴더에 일단 그대로 넣습니다.
2. 명령 프롬프트에서 `python convert.py` 를 실행하면 자동으로 WebP 로 변환됩니다.

### 방법 ② — 온라인 변환기
- `https://squoosh.app` 또는 `https://convertio.co/jpg-webp/` 등
- 변환 후 파일 이름을 위 표의 이름으로 바꿔서 이 폴더에 저장합니다.

### 방법 ③ — Windows / 모바일 기본 도구
- 갤러리 앱에서 "공유 → 다른 형식으로 저장" 으로 WebP 선택이 가능한 폰도 있습니다.

---

## 사진을 갤러리에 추가/교체하려면

`data.js` 의 `gallery: [...]` 부분에서 `src` 값(파일 경로)을 바꾸거나,
관리자 페이지(`admin.html`)에서 "③ 사진 갤러리" 카드를 통해 편집하시면 됩니다.
